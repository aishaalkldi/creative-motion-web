# RASQ Decision Log

## Permanent Decisions

- RASQ is a rehabilitation intelligence platform, not a single assessment application.
- Clinical Safety is governance, not a product domain.
- XR Rehabilitation is a strategic product domain.
- Communication Intelligence is a strategic product domain.
- Measured values remain separate from AI interpretation.
- AI output is assistive and non-diagnostic.
- GitHub is the source of truth.
- Development does not happen directly on main.
- Prefer small reversible changes over broad refactors.
